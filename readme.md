
# Meta Tic Tac Toe

Meta Tic Tac Toe with Monte Carlo Tree Search (without neural networks).

```
+-1-2-3-+-4-5-6-+-7-8-9-+
1       | o o o |       |
2       | o o o |       |
3     o | o o o | o   o |
+-------+-------+-------+
4   x   |     x |   x   |
5       |   x   |       |
6       |     o |       |
+-------+-------+-------+
7 x     |       | . x x |
8       |       | . . . |
9       |       | . . . |
+-------+-------+-------+
        x to move
        
Choose your move!

```
